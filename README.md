# CS-530 AI Tool Evaluation Project

## Course
CS-530: Artificial Intelligence Applications

## Project Title
GitHub Copilot Evaluation for Software Development

## Description
This project evaluates the use of GitHub Copilot as an AI-powered
code generation tool within a simulated enterprise environment.

The evaluation focused on prompt engineering, debugging,
code generation, and ethical considerations related to
AI-assisted programming.

## Tools Used
- GitHub Copilot
- Python
- Natural Language Prompts
- JSON Chat Logs
- Microsoft Word

## Project Tasks

### Part One: AI Tool Evaluation
- Created initial prompts for code snippets
- Refined prompts to improve outputs
- Debugged provided code using AI
- Generated new code from functional requirements
- Tested generated code functionality

### Part Two: Evaluation Summary
- Explained machine learning and deep learning relationships
- Evaluated natural language processing in AI tools
- Analyzed ethical risks including bias and privacy
- Assessed long-term impacts of AI-assisted development

## Skills Demonstrated
- Prompt Engineering
- AI Code Evaluation
- Debugging with AI Tools
- Ethical Risk Assessment
- Software Development Analysis
